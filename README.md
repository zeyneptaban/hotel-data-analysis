# Hotel Data Analysis

Exploratory data analysis project based on hotel booking datasets.

## Objectives
- Analyze booking trends
- Study cancellation rates
- Explore customer segments
- Visualize hospitality data

## Tools Used
- Python
- pandas
- matplotlib
- Excel

## Key Insights
- High cancellation rates impact hotel forecasting
- Online travel agencies generate most reservations
- City hotels receive significantly more bookings
